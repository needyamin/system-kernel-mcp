Install
python -m venv venv

MCP Config (Cursor / Claude)
{
  "mcpServers": {
    "SystemKernelMCP": {
      "command": "python",
      "args": ["${workspaceFolder}/system_mcp.py"],
      "cwd": "${workspaceFolder}"
    }
  }
}

Use absolute path if workspaceFolder fails: args: ["C:\\path\\to\\system_mcp.py"]

Use Cases
  • Pentesting: Port scan, DNS, SSL cert, JWT decode, network enum
  • Automation: Commands, file ops, batch scripts, env vars
  • DevOps: Docker, Kubernetes, VM control
  • System: BIOS, kernel, modules, boot params
Requirements

Python 3.10+

Optional: docker, kubectl, VirtualBox, VMware, Hyper-V, WSL, libvirt
