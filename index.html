SystemKernelMCP

Documentation

Cross-platform MCP server for terminal, OS info, and processes.

Tools
ToolDescription
execute_terminal_commandRun shell commands (timeout 300s)
run_commandsRun multiple commands in sequence
get_os_infoOS, kernel, arch, Python, CWD
get_running_processesTop processes by CPU
get_current_userUsername and UID
read_fileRead file contents
list_directoryList files/folders
write_fileWrite or append to file
file_existsCheck if path exists
ping_hostPing a host
check_portCheck if TCP port is open
get_network_infoIP addresses, interfaces
get_environment_varsList env vars
check_tool_installedCheck if CLI tool exists
get_servicesRunning services
Install
python -m venv venv
pip install mcp
MCP Config (Cursor / Claude)
{
  "mcpServers": {
    "SystemKernelMCP": {
      "command": "python",
      "args": ["${workspaceFolder}/system_mcp.py"],
      "cwd": "${workspaceFolder}"
    }
  }
}
Use Cases
  • Pentesting: Port checks, ping, network info, process/service enumeration
  • Automation: Run commands, batch scripts, file ops, env vars
  • Non-technical: Plain-language tool descriptions, sensible defaults
Platforms
Windows Linux macOS
